Нормальный синтезатор с возможностью автоматически проигровать ноты







t: 0:00.00 / 0:00.00

* Режим "Воспроизведение (зажатие / переключение)" — по клику звук вкл/выкл (можно нажать несколько кнопок одновременно)
* Режим "Воспроизведение (только пока удерживаешь)" — звук звучит только пока кнопка нажата (мышь или палец), поддержка одновременных удержаний
* Режим "Редактирование" — по клику можно менять частоту и цвет кнопки
* Режим "Удаление" — по клику удаляется кнопка
* При смене режима все звуки прерываются
* Кнопки автоматически сортируются по возрастанию частоты при добавлении и изменении
* Частоты можно задавать от 1 до 20000 Hz
* Можно менять форму волны для всех играющих звуков
* Правый клик и двойной клик отключены для предотвращения ошибок
* В текстовом поле можно записать мелодию которая будет сама воспроизводиться по кнопке "Воспроизвести"
Структура записи такая: сначала пишется частота ноты, затем через пробел её длительность в сантиударах при темпе 120bpm
где тот самый b из bpm это и есть удар, а затем снова через пробел сдвиг начала ноты в тех-же сантиударах относительно конца предыдущей ноты
Сдвиг кстати писать не обязательно и по умолчанию он = 0, а через перенос строки записывается слудующая нота 🤷🏻‍♂️
